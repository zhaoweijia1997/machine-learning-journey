# 👤 人脸识别系统

> 基于 OpenVINO 的人脸识别，支持人脸录入和实时识别

## ✨ 功能特性

- 🔍 **人脸检测**: face-detection-retail-0005 模型
- 🎯 **人脸识别**: face-reidentification-retail-0095 模型
- 📷 **双模式**: 摄像头 / 屏幕捕获
- 💾 **数据库**: GUI 管理人脸数据

## 📁 目录结构

```
face-recognition/
├── face_app.pyw        # 🚀 GUI 应用（推荐）
├── face_register.py    # 人脸数据库模块
├── 启动人脸识别.bat     # 快捷启动
│
└── face_db/            # 📦 数据库目录
    ├── embeddings.pkl  # 特征向量
    ├── face_info.json  # 人脸信息
    ├── photos/         # 录入照片
    └── models/         # OpenVINO 模型
```

## 🚀 快速开始

```bash
# 双击运行
启动人脸识别.bat

# 或命令行
python face_app.pyw
```

## 📝 使用说明

### 录入人脸

1. 点击 **"摄像头"** 启动
2. 点击 **"开始录入"** 输入姓名/职称
3. 面向摄像头，按 **S** 键拍照
4. 建议录入多角度照片提高准确率

### 识别人脸

1. 选择 **"摄像头"** 或 **"屏幕捕获"** 模式
2. 系统自动识别画面中人脸
3. 已录入人员显示姓名和置信度

## ⌨️ 快捷键

| 按键 | 功能 |
|------|------|
| **S** | 拍照录入 |
| **ESC** | 停止 |

## 🔧 技术细节

- 256维特征向量表示人脸
- 余弦相似度匹配
- 模型首次运行自动下载
